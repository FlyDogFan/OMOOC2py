#Mydaily-Wechat
##Introduction
- 通过微信公共账号输入笔记, 基于SAE对笔记进行保存以及相关的极简交互.

##Installation
- 申请微信公共账号
- SAE云平台
- 将`index.wsgi`以及`config.yaml`部署到SAE云平台
- 在微信公众账号后台进行token确认.具体才能见[教程]()
- 准备工作完毕后, 即可在公众账号中记录笔记.
